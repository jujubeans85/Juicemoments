# JUICE MOMENTS

A tiny reusable reveal engine for gifts, NFC tags, QR codes and hidden links.

## Live priority moment

- Chloe coffee gift
- Root route: `/`
- Named route: `/m/chloe-coffee`
- Query route: `/?moment=chloe-coffee`

## Add another moment

1. Add its image to `/assets/`
2. Copy `/moments/chloe-coffee.json`
3. Change the text, media, timing and destination
4. Register it in `/moments.json`

One engine. One theme. Many moments.
